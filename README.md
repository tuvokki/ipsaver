# Mereorized ipsaver

> Save your ip, just because it's possible. This time with the Meteor framework

## Install
- clone
- Set the mongo or use locally
- meteor
- open url

## test
from mongo console: db.iphits.insert({host:"test",ip:"127.0.0.1",msg:"mongo",date:Date.now()})
